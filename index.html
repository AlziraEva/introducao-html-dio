Estrutura Básica do HTML

Voltar

Temos três estruturas básicas no HTML, que são as tags: Html, head e body

Html

Tudo que está dentro dessa tag se refere a página HTML.

Head

Onde fica localizado tudo aquilo que não vemos na página web.

Body

Onde está toda a parte que podemos ver.

Falando sobre Tags

Voltar

Tudo que está dentro das tags o navegador vai interpretar de uma forma diferente para cada tag.

Tags para textos

  • i : O texto fica em ítalico.
  • strong : O texto fica em negrito.
  • i : O texto fica em ítalico.

Tags em geral :

  • input : Para gerar uma caixa de texto.
  • img : Inserir uma imagem.

Atributos das Tags

Voltar

São propriedades que as tags podem ter.

Atributos genéricos

  • Id : Identificador do elemento.
  • Style : Voltado para colocar comandos do CSS (Forma de formatar o HTML)
  • Class : Forma de padronizar várias tags para uma formatação só.

Atributos especificos

  • Type : Atributo especifico da tag Input.
  • Src : Atributo especifico da tag Img., onde é colocado dentro dele o caminho onde se encontra a imagem.
  • Width : Atributo especifico da tag Img., define a largura da imagem.

Textos

Voltar

As principais tags utilizadas para textos :

  • h1 : Utilizada para o titulo principal.
  • h2 a h6 : Utilizadas para subtitulos.
  • p : O corpo do texto, cada quebra de linha ele é utilizado.
  • blockquote : usado para citações.
  • u : Para sublinhar.
  • mark : marcar texto.
  • sub : colocar o texto na linha superior.

Listas

Voltar

Existem dois tipos de listar são elas: Ordenadas e não ordenadas.

Ordernadas

São listas que seguem uma sequencia, exemplo: 1. 2. 3. ...., sendo representada pela tag ( ol ) e dentro dela temos as tags filhas ( li ).

Não Ordenadas

São listas que não são sequenciadas, sendo representada pela tag ( ul ) de dentrio dela as tags filha ( il ).

Links

Voltar

Utilizada para ligar uma página na outra, representada pela tag ( a )

Atributos

  • href : Onde é colocado o caminho do link.
  • target : Para definir o comportamente ao clicar no link, podendo ser do tipo ( _self ) que ao clicar no link é feita a abertura na mesma aba, e o ( _bland ) que o link redireciona para outra aba.
  • title : Ao passar o mouse em cima do link, aparece uma caixa com o texto definido no title.
